# Modelagem Preditiva da Rotatividade de Funcionários com Machine Learning
**Universidade do Estado do Rio de Janeiro (UERJ) – 2025**

## Sobre
Projeto de Ciência de Dados aplicado à previsão de rotatividade de funcionários, desenvolvido ao longo da graduação. O trabalho cobre o pipeline completo: definição do escopo, preparação e limpeza dos dados, análise exploratória, engenharia de features, modelagem e avaliação.

## Ferramentas e bibliotecas
- Python  
- Pandas, NumPy, Scikit-Learn, Matplotlib  
- Ambiente: Google Colab

## Etapas realizadas
1. **Definição do escopo:** formulação do problema e objetivos.  
2. **Preparação e limpeza:** tratamento de valores ausentes, correções de tipos e formatação.  
3. **Análise exploratória:** visualizações e estatísticas para identificar padrões.  
4. **Modelagem e avaliação:** treinamento e validação de modelos supervisionados, comparação de desempenho e interpretação dos resultados.
